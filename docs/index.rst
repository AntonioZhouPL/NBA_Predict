NBA Predict Documentation
=========================

NBA Predict is a reproducible Python pipeline for predicting NBA regular season
game outcomes. The active implementation lives in the ``nba_predict`` package
under ``src/`` and is designed to be run through the ``nba-predict`` command
line interface or the repository ``Makefile``.

The documentation generated by Sphinx covers the package modules used for data
download, preprocessing, model evaluation, reporting, and deterministic
reproducibility checks.

Build the HTML documentation from the repository root with:

.. code-block:: bash

   make docs

The generated HTML files are written to ``docs/_build/html``.
